Prior to running, make sure nodejs and express are installed:
Install nodejs: https://nodejs.org/en/download/package-manager/
Install express: https://expressjs.com/en/starter/installing.html


To Run:

1. Navigate to directory containing the app.js file.
2. Run the command: node app.js 
3. Navigate to http://localhost:8080 in your web browser. 

** To change input file, open app.js and change the readfile name on line 22. **
